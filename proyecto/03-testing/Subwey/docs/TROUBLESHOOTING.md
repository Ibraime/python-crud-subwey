# Troubleshooting

Errores comunes y posibles causas/soluciones.

---

- "El nombre no existe": se intentó modificar o eliminar un nombre inexistente.
- "El precio no puede ser menor que 0": precio invalido en alta.
- "No se puede consumir: no hay suficiente stock.": se intentó consumir más stock de el que hay.
- "No se permiten ingredientes repetidos": se intentó poner el mismo ingrediente más de 1 vez.
- "Ya existe un bocadillo con ese nombre.": se intentó crear un bocadillo con el mismo nombre

- "El descuento debe estar entre 1 y 90.": porcentaje fuera de 1..90.
