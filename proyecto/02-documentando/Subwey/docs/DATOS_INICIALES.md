# Datos iniciales

Aquí se incluyen:
- Que datos carga por defecto la aplicación al iniciarse..
- Como modificar los datos que se cargan por defecto para pruebas/demos.

---

## Datos precargados
La funcion `crear_repositorio_con_datos()` carga:
- A1: Agua (1.00, 10)
- A2: Papas (1.50, 8)
- D1: Refresco con descuento (2.50, 5, 20%)

## Modificar datos
Editar `infrastructure/datos_iniciales.py` para:
- Cambiar nombres/precios.
- Ajustar cantidades.
- Agregar nuevos items o descuentos.
